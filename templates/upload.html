Import

You can import your existing bookmarks with this tool. All bookmarks will be saved into a new list with tags and descriptions as notes.

  1. Go to Google Bookmarks
  2. Click "Export bookmarks"
  3. Upload the GoogleBookmarks.html file
  1. Go to Bookmark Manager
  2. From the Organize menu choose "Export bookmarks to HTML File..."
  3. Save the file on your computer
  4. Upload the saved .html file
  1. Go to Bookmarks › Show All Bookmarks ()
  2. From the "star" menu choose "Export HTML"
  3. Save the file on your computer
  4. Upload the saved bookmarks.html file
  1. Go to File › Export bookmarks
  2. Save the file on your computer
  3. Upload the saved Safari Bookmarks.html file
